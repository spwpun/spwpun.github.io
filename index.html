使用KMS激活Office

KMS技术是通过Key Manager Service服务的激活机制来激活Windows或者office的，能获取正版的产品密钥，
不过密钥有有效期，不是永久的，所以等到有效期到了之后需要重新获取密钥。相比网上的一些破解软件来
说要好的多。

  1. 打开最新的KMS Tools by Ratiborus,会出现多个应用选项。
  2. 点击”KMSoffline”,进入KMSoffline x64 v2.1.3界面,有Main、Tools andSettings、About四个菜单选项。
  3. 在下方”Product”处，可以选择激活Windows和office,当然office需要自己安装。
  4. 然后点击右下方的”Information”可以查看Windows的版本信息或者office的版本信息，查看是否激活，有效期（）等什么的：
    例如：下方是Windows的信息
    == Windows(R), ProfessionalWorkstation edition
    Windows(R) Operating System, VOLUME_KMSCLIENT channel
    SKU ID: 82bbc092-bc50-4e16-8e18-b74fc486aec3
    Partial Product Key: 6Q84J
    KMS-host Machine: 
    KMS-Server PID: 03612-00206-557-633704-03-2052-17763.0000-1232019
    Expiration Date/Time: 179.20:24, until 2020/4/11 //这里显示到2020/04/11过期。
    Activation interval: 00.02:00
   Renewal interval: 16.04:23
       License Status: --- LICENSED ---
---------------------------------------
  5. 所以”Product”选择office，查看信息，确定office版本，点击”Install Key”,生成产品密钥，中间的白框框中就会出现提示信息，
    最后的是密钥，例如：
    ---------------------------------------
    == Installation GVLK keys Office 19
    Install License:
    ProPlus2019VL_KMS_Client_AE-ppd.xrm-ms
    Install License:
    ProPlus2019VL_KMS_Client_AE-ul-oob.xrm-ms
    Install License:
    ProPlus2019VL_KMS_Client_AE-ul.xrm-ms
    Install License:
    ProPlus2019VL_MAK_AE-pl.xrm-ms
    Install License:
    ProPlus2019VL_MAK_AE-ppd.xrm-ms
    Install License:
    ProPlus2019VL_MAK_AE-ul-oob.xrm-ms
    Install License:
    ProPlus2019VL_MAK_AE-ul-phn.xrm-ms
    Uninstall key: 8MBCX
